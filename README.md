# Spotify MusicSaver
Spotifyで聞いた曲の履歴を保存するやつ

保存した履歴からプレイリストを使うなり、データ解析の素材にするなり

## 1. pipenvを使う場合
pipenvが必要です。
```sh
$ pip install pipenv
```
インストール後は同様に、以下のコマンドを実行してください。
```sh
$ pipenv install
$ pipenv shell
```

起動するにはmain.pyを実行してください
```sh
$ python main.py
```
